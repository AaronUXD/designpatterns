Fonts

Internal/private applications

Font load and fallback order: Calibri, Helvetica Neue, Helvetica, Arial, sans-serif

External/public applications

Font load and fallback order: Helvetica Neue (web font), Helvetica, Arial, sans-serif

Font availability

Calibri
Ships with Windows Vista and newer. Ships with Office for Mac. Calibri Light available on Windows 8+.
Helvetica Neue / Helvetica
Ships with OSX.
Arial
Ships with most operating systems.
