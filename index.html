What is this?

An open source design guide, pattern library, and Bootstrap theme. Our docs are packed full of usability guidelines, examples, and source code.

How can I use it?

Once downloaded from the link above or via Bower, you can use either the minified CSS build or the source LESS files in your own build system.

Who is it for?

For designers and developers working on USPTO related projects, but since it's public domain anyone can use it. If you can benefit from it, use it.

Why did we make it?

We have a lot of web projects at the USPTO. These design guidelines and resources aim to create a unified experience across all projects at the USPTO, both internal and external.

Not using Bootstrap?

That's fine! A lot of our projects use Bootstrap, but not all of them. The usability guidelines and some resources (like LESS variables or icons) could still be useful.

Want to contribute?

We want your help! Visit our Github repository for contribution guidelines. If you have improvements or bug fixes, let us know!
